网络​

内网穿透(ngrok)​

  1. 登录 https://dashboard.ngrok.com 完成账号注册（不支持 qq 邮箱，可以用其它替代，如 163.com）
  2. 角色选择软件工程师，会弹出 ngrok 的安装教程，执行相应命令完成安装
  3. 执行安装：brew install ngrok，若终端提示To install shell completions, add this to your profile ，则将下列代码添加至 ~/zshrc 文件中
if command -v ngrok &>/dev/null; then
    eval "$(ngrok completion)"
fi
  1. 配置身份 token: ngrok config add-authtoken <your_token>
  2. 配置需要穿透的本地或內网 ip 地址（同时支持 http/https），例如 ngrok http http:/localhost:8080
  3. 穿透成功，可以访问web Interface可视化查看公网访问记录
